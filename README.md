## ▸ Featured Projects

### [zeroentropy-rust](https://github.com/davidatoms/zeroentropy-rust)
Rust SDK for the ZeroEntropy semantic search API, with Brain-Computer Interface phoneme-to-word matching capabilities.

**Highlights:**
- Complete API implementation with async/await
- Tested on 1,718 BCI dataset phoneme-word pairs
- 40% success rate at scale, 100% with smaller datasets
- Documentation for RNN+n-gram integration strategies

**Stack:** Rust, Tokio, Reqwest, Serde
